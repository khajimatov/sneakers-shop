# Sneakers Shop

Project build with ReactJS. I am developing this website to add it to my portfolio and learn HTML, CSS, JavaScript and ReactJS throughout development and learning process.

## Functionalities:

- Cart
- Favorites
- Orders
- Works with database(MockAPI)
- Search
